## 快捷键

Prefix 为 `Ctrl+a`（按下后松开，再按对应键）。

| 快捷键 | 功能 |
|--------|------|
| `prefix c` | 创建新 window |
| `prefix n` | 下一个 window |
| `prefix p` | 上一个 window |
| `prefix X` | 关闭当前 window |
| `prefix d` | detach 当前 session |
| `prefix \|` | 左右分屏 |
| `prefix -` | 上下分屏 |
| `prefix x` | 关闭当前 pane |
| `prefix h/j/k/l` | 切换 pane（vim 风格） |
| `prefix r` | 重载配置 |
